# C# Fundamentals

A collection of C# fundamentals and algorithm implementations focused on building a strong foundation in problem solving and core language concepts.

## Topics covered
- Core C# syntax and language features
- Control flow and data structures
- Algorithmic problem solving

## Algorithms
- Binary Search
- Depth-First Search (DFS)
- Breadth-First Search (BFS)
- Sliding Window techniques

## Purpose
This repository is used to practice and document fundamental C# concepts and common algorithms that are frequently used in backend and application development.
